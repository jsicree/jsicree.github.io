# jsicree.github.io

Joe Sicree's personal GitHub Pages homepage.

## Overview

A Jekyll-based personal homepage served via GitHub Pages. Jekyll is processed automatically by GitHub Pages — no local build step required.

## Files

- `index.html` — main page (Jekyll front matter + Liquid templating)
- `style.css` — stylesheet
- `_config.yml` — Jekyll site configuration
- `_layouts/default.html` — base HTML layout template

## Deployment

Pushes to the `master` branch are automatically deployed at [jsicree.github.io](https://jsicree.github.io).
